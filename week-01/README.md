# Week 1: Introduction to Machine Learning

This week covers the ML project lifecycle, data types, the main types of ML systems, and the tools used throughout the course.

## How to work through this week

1. In `ml-course`, run `git pull` to get this week's files (this repo is read-only for you).
2. In your own repo `ml-course-labs`, create a `week-01/` folder.
3. Copy this week's README and homework files into it:
   ```
   cp ../ml-course/week-01/README.md  week-01/
   cp ../ml-course/week-01/*.py       week-01/
   ```
4. Solve the homework below.
5. Commit and push to your own repo only. Never push to `ml-course`.

## Homework

- `01_hw_scores.py`: pure-Python student scores exercise (built-in Python only)
- `02_hw_library.py`: pure-Python library management system (built-in Python only)
